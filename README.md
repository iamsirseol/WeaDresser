<h2>서비스 설명</h2>
현재 유저의 위치와 그 위치에 따른 날씨 정보 조회<br />
조회 한 데이터를 통해 우리가 유저가 오늘 날씨에 어울리는 코디 추천<br />
유저가 오늘 입은 옷과 일기를 작성이 가능<br />
